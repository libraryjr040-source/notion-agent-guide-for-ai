# 《Notion Agent 使用指南》

AI-first 操作性知识库，由多 Agent 协作系统自动编写。

## 项目概述

本书面向 AI Agent（而非人类读者），是一本关于如何在 Notion 平台上高效操作的指南。内容涵盖概念索引、内容操作、数据操作、Agent 工程、多 Agent 架构、踩坑与优化六大章节。

## 编写团队

- **总编**：全流程编排、context 管理、质量闭环
- **著作郎**：章节写作、技术验证、draft 提交
- **校勘官**：逐条验证技术准确性、场景覆盖度、独立可加载性
